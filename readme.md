# Setup system

```
git clone git@github.com:drujensen/dotbash.git .bash
ln -sf .bash/gitconfig .gitconfig
ln -sf .bash/git_template .git_template
```

## bash
```
ln -sf .bash/bash_profile .bash_profile
```

## zsh
```
ln -sf .bash/zshrc .zshrc
```
